A few of my goals for this site

As much as I know this is a school assignment, I wanted to be able
to use this as a jumping off point for a complied collection of my work.
An interactive CV if you will. It may be bare bones for the moment, but
I'm hoping I can expand on it with more time, experience and achievements!
For now, my goals for this site are as follows:

>

Use the Site for learning

I want to try to get a solid grasp on how both uploading and managing an active site works. I want to understand how to upload content to a site and alter it. I think I will struggle with this the most.

Use the site for documentation

I have a lot of active projects going on at the moment! Even if I never return to this site for the future, I want to be able to look at it and remember all the things I worked on, and maybe know more about where they went.

Use the site as a base

Ideally, I'd love to learn enough about using this site so that I can actively use this as an interactive CV in the future so that either employers or potentiall clients can see what I'm working on.
